How the System Works

Our system uses a deep-learning powered pipeline to predict a user’s blood group from fingerprint patterns. The entire workflow is fully automated, fast, and follows a structured sequence of processing steps to ensure accuracy and reliability.

Step-by-Step Process

  1. Upload Fingerprint Image
    The user uploads a fingerprint photograph or scanned image in JPG/PNG format.
  2. Image Preprocessing
    The server resizes the image to 256 × 256, normalizes pixel values, and applies the preprocessing pipeline used during training.
  3. Feature Extraction using VGG16
    A pre-trained VGG16 CNN extracts deep fingerprint features such as ridges, bifurcations, and unique ridge flow patterns.
  4. Classification using Custom Layers
    The extracted feature vector is passed to dense layers that classify the fingerprint into one of the 8 possible blood groups.
  5. Prediction Output
    The predicted blood group and confidence percentage are displayed to the user.
  6. Saving the Results (For logged-in users)
    The prediction details are securely stored in the MySQL database for future reference.
  7. Real-Time Access Anywhere
    Users can review prediction history, accuracy trends, and upload new fingerprints.

Why This Works

Convolutional Neural Networks like VGG16 are highly effective at learning complex visual structures. Fingerprint ridge patterns contain subtle correlations with biological traits, including blood groups. The deep-learning pipeline captures these patterns and produces reliable predictions.

Technologies Used

  • VGG16 – Transfer Learning
  • Keras
  • Image Preprocessing & Normalization
  • Flask Web Framework
  • MySQL Database
